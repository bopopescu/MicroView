=========================================
MicroView 3D Image Viewer & Analysis Tool
=========================================

.. raw:: html

  <iframe width="461" height="300" align="right"

   </iframe>

MicroView is an open source, dual-licensed, 3D image viewer. It includes many image analysis
tools and plug-ins. It can be installed on most computers running Windows XP or later.
A version for select Linux distributions and Mac OS X systems is also available.

For more information please contact us at info@parallax-innovations.com.
